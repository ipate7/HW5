# HW5

Repository: [github.com/ipate7/HW5](https://github.com/ipate7/HW5)

After you enable GitHub Pages (Settings → Pages → branch **main**, folder **/ (root)**), the site will be at:

**https://ipate7.github.io/HW5/**

That URL is what you paste into PrairieLearn for the Jekyll submission. The Analysis button on the site points to:

**https://github.com/ipate7/HW5/blob/main/Workbook.ipynb**

`_config.yml` already sets `url` and `baseurl` for this project-site layout.

Push the contents of this folder to the repo root, then commit everything listed below (not `.venv/` or local `_site/`).

| Item | Purpose |
|------|--------|
| `_config.yml`, `index.md` | Jekyll site |
| `assets/` | Chart HTML and JSON for the page |
| `Workbook.ipynb` | Notebook |
| `requirements.txt`, `.gitignore` | Optional / hygiene |

Re-run `Workbook.ipynb` after plot changes so `assets/` stays current before you push.
